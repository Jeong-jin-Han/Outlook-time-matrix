Finished Category

Select which category color marks an event as finished in Outlook.

How to use

Load Restore saved placements. Past events → Finished. New events → ToDo.
Refresh Reset ALL placements to ToDo (asks for confirmation first).
🗓 Open Outlook Calendar in the browser tab.
⊞ / ≡ Switch between 2×2 grid view and 4×1 list view.
◀ / ▶ Expand or collapse the ToDo / Finished sidebar.
Drag & drop Move tasks between any quadrant or the Common sidebar.
☑ Checkbox Instantly mark a task as Finished (uncheck to move back to ToDo).
